getWVToken

获取webview本地存储数据

1.WVJsBridge.getToken(CALLBACK)

名称 类型 必填 描述
callback function 是 回调函数拿到数据

2. 代码示例

ES6箭头函数形式

WVJsBridge.getToken((data)=>{
     const token = data
})

ES5 普通回调函数形式

WVJsBridge.getToken(function(data){
    var token = data
})
