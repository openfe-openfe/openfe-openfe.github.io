介绍

潍V H5开放文档

WVJsbridge是潍V客户端通过brdige，允许前端H5页面通过特定的JS方法，可以直接调用对应的支付、拍照、分享、弹出浮层之类的Native功能。

如何使用

1.直接下载 wvbridge sdk 页面直接引入

  <script src="wvbridge"></script>

2 或者npm install wvbridge --save 导入到项目中

import WVJsBridge from 'wvbridge'
