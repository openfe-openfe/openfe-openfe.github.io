关于react-native run-ios 运行提示network request filed的解决办法

发表于 2016-10-20 | 分类于 react-native

```
关于运行提示network request filed解决方案：
这个需要我们在xcode里面操作。

找到info.plist文件 ，这里需要添加Allow Arbitrary Loads（一定是在App Transport Security Settings里面添加 ）类型是布尔值 并且设置为true。

我的RN版本是.33版本 ，并没有带这个，所以会出现问题，需要我们手动添加。

另外如果按照老的文章，添加NSAppTransportSecurity 会提示覆盖（千万别覆盖，因为他会重新生成一个新的App Transport Security Settings）， 里面都是空的。倘若你覆盖了，则需要重新添加Exception Domains->localtion->NSTemporaryExceptionAllowsInsecureHTTPLoads
